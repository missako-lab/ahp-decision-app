# AHP Decision Support System

## Description
Application web basée sur la méthode AHP pour la prise de décision multicritère.

## Installation

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start

## Utilisation
- Remplir la matrice AHP
- Ajouter scores
- Cliquer sur Compute

## Résultat
- Poids des critères
- Ratio de cohérence (CR)
- Classement des alternatives
